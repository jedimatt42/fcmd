##  Introduction

This is a simplistic device to read the contents of GROM only (one
sidedPCB) Command Modules for the TI-99/4A using the parallel port of a
standardPC.

Parts needed:

- One 2x18 pin female 2.54 mm card edge connector (you can use the
  narrowend of an ISA connector)
- One 25 pin male D-SUB connector for the parallel port
- One 6 pin male Mini-DIN connector for the keyboard/mouse port (usedfor
  power supply)
- One EXO3 oscillator at 12.8MHz
- One LMC7660 voltage inverter
- Three 10µF electrolytic capacitors
- One 4.7k? resistor
